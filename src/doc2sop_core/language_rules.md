# Language Rules for SOP Humanization

## Remove AI Voice
- No "As an AI..." or "I would recommend..."
- No "This document explains..."
- No "In conclusion..."
- No promotional language

## Tone
- Boring, professional
- Clear action verbs
- No emoji
- No casual filler words

## Grammar Fixes
- "Id" -> "I'd"
- "Its" -> "It's" (when appropriate)
- "dont" -> "don't"
- "gonna" -> "going to"
- "gotta" -> "need to"

## Preserved Elements
- Keep must/should/may as written
- Preserve numbered steps exactly
- Don't change meaning even if awkward
